中华传统文化之对联

风俗| 历史| 对法| 张贴| 实例

项目：拾趣 妙对

春联，又称“春贴”、“门对”、“对联”、“桃符”，是过年时所贴的红色喜庆元素“年红”中一个种类。它以对仗工整、简洁精巧的文字描绘美好形象，抒发美好愿望，是中国特有的文学形式，是华人们过年的重要习俗。 贴对联是春节的标志性礼俗之一，表达了中国劳动人民一种辟邪除灾、迎祥纳福的美好愿望。 据说这一习俗起于宋代，在明代开始盛行，到了清代，春联的思想性和艺术性都有了很大的提高。现在贴春联已成风俗，红色的对联贴在大门上，房子顿时生辉。正如诗云：“喜气临门红色妍，家家户户贴春联；旧年辞别迎新岁，时序车轮总向前。”
